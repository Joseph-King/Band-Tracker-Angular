Welcome to Band Tracker

This site was created by Joseph King to allow users to have an account and track artists, venues, shows, and their own personal list of shows that they have been to.

The website is still in production and further features will be added in the future. Current we are on version 0.2.1

This website is a MEAN stack application and uses MongoDB, Express, Angular, and Node.
